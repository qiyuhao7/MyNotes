# Markdown学习

## 标题

```markdown
# 一级标题
## 二级标题
```

## 字体

```markdown
*Hello* 斜体
**hello** 黑体
***hello*** 斜黑体
~~删除线~~
```

## 引用

```markdown
> 表示引用
```

## 分割线

```markdown
*** 三个星号表示分割线
--- 表示分割线
```

## 图片

```markdown
![标题](地址)来显示图片
图床可以使用sm.ms
```

## 超链接

```markdown
使用[标题](地址)来表示超链接
```

[sm.ms](https://sm.ms)

## 列表

```markdown
1. A 表示有序的列表
- A 表示无序的
```

1. A

2. B

- A
- B
